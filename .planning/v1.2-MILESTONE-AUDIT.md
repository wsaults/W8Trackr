### Translation Gaps from Earlier Releases

17 strings in main app lack translations for 8 new languages:

**Milestone-related (4 strings):**
- `"%.1f %@ to next milestone"`
- `"Milestone %lld %@, achieved %@"`
- `"Milestone reached! You've hit %lld %@. Keep up the great work!"`
- `"Next milestone %lld %@. %lld percent complete. %@ %@ remaining."`

**Goal display (2 strings):**
- `"Goal: %.0f %@"`
- `"Goal: %@ %@"`

**Export feature (1 string):**
- `"Export entries from %@ to %@"`

**Generic format strings (8 strings):**
- `"%@ (%@)"`, `"%@ %@"`, `"%@ %@ this week"`, etc.

**Impact:** Low. These strings show in English when device is set to new languages. All core weight tracking features are fully translated.

**Recommendation:** Address in v1.3 patch release or future localization pass.
