# W8Trackr - Spanish App Store Metadata

## App Name (30 characters max)
**English:** W8Trackr - Weight Tracker
**Spanish:** W8Trackr - Control de Peso

## Subtitle (30 characters max)
**English:** Track your weight journey
**Spanish:** Sigue tu progreso de peso

## Promotional Text (170 characters max)
**English:** Simple, beautiful weight tracking with smart trends and milestone celebrations.
**Spanish:** Seguimiento de peso simple y elegante con tendencias inteligentes y celebraciones de logros.

## Description (4000 characters max)

**Spanish:**
W8Trackr te ayuda a seguir tu progreso de peso con una interfaz hermosa y facil de usar.

CARACTERISTICAS PRINCIPALES

- Registro rapido de peso con soporte para libras o kilogramos
- Graficos de tendencia con analisis EWMA para ver tu progreso real
- Celebraciones de hitos cuando alcanzas tus objetivos
- Sincronizacion con iCloud para respaldo automatico
- Integracion con Apple Salud para importar y exportar datos
- Widgets para tu pantalla de inicio
- Recordatorios diarios configurables
- Exportacion de datos en CSV o JSON
- Modo oscuro completo
- Accesibilidad WCAG 2.1 AA

SEGUIMIENTO INTELIGENTE

W8Trackr utiliza el algoritmo EWMA (Exponentially Weighted Moving Average) para suavizar las fluctuaciones diarias y mostrarte tu tendencia real de peso. No mas estres por variaciones normales dia a dia.

PRIVACIDAD PRIMERO

Tus datos permanecen en tu dispositivo y en tu iCloud personal. No recopilamos ni compartimos tu informacion de peso con nadie.

SINCRONIZACION CON APPLE SALUD

Importa tus entradas de peso existentes de Apple Salud y exporta nuevas entradas para mantener todos tus datos de salud sincronizados.

WIDGETS

Agrega widgets a tu pantalla de inicio para ver tu peso actual, progreso hacia tu objetivo, o un grafico de tendencia de un vistazo.

Comienza tu viaje de peso hoy con W8Trackr.

## Keywords (100 characters max, comma-separated)
**English:** weight,tracker,scale,diet,fitness,health,loss,gain,goal,trend
**Spanish:** peso,seguimiento,bascula,dieta,salud,perdida,aumento,objetivo,tendencia,control

## What's New (4000 characters max)
**Spanish:**
Version 1.1:
- Soporte para idioma espanol
- Importacion de datos de Apple Salud
- Widgets para pantalla de inicio
- Compartir imagenes de progreso
- Mejoras de rendimiento

   - Set device language to Spanish (Settings > General > Language & Region)
   - Launch W8Trackr and verify:

     - [ ] Dashboard labels are in Spanish
     - [ ] Settings screen is fully translated
     - [ ] Weight entry dialog is in Spanish
     - [ ] Onboarding flow (if accessible) shows Spanish
     - [ ] Milestone celebration shows Spanish text

2. **Test number/date formatting (LOCL-02):**
   - [ ] Verify weight numbers display correctly (decimal separator should be comma in Spanish locale: 75,5 kg)
   - [ ] Verify dates display in Spanish format (10 de enero de 2026)

3. **Test widgets:**
   - [ ] Add small widget - verify "Tendencia al alza/baja" text
   - [ ] Add medium widget - verify "Progreso al Objetivo" text
   - [ ] Verify widget gallery shows Spanish name/description

4. **Test permission dialogs:**
   - Reset permissions (delete app, reinstall)
   - Verify HealthKit permission shows Spanish description
   - Verify notification permission shows Spanish description

5. **Test Siri (optional):**

   - Verify Siri understands the Spanish phrase

6. **Review App Store metadata:**
   - Open .planning/phases/25-localization/APP_STORE_SPANISH.md
   - Verify translations sound natural to Spanish speakers
   - Flag any awkward phrasing
